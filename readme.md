## Getting Started

1. Install RNC Styles using npm:

   ```bash
   npm install rnc-styles
   ```

2. Import and use styles in your React Native project:

   ```javascript
   import RNCStyles from 'rnc-styles';
   // Use styles from RNCStyles in your components
   ```
